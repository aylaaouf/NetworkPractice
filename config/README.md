# NetPractice

## üìò Description
NetPractice is an interactive networking training project from the 42/1337 curriculum.  
Its goal is to teach the fundamentals of computer networking through a series of 10 exercises.  
Each level contains a broken network topology that must be fixed by correctly configuring:

- IPv4 addresses  
- Subnet masks  
- Default gateways  
- Routing tables  

The objective is to understand how packets move through a network and how devices communicate across different subnets.
